# Bootcamp Intensivo de Python da Jornada de Dados

Repositório relacionado aos meus estudos da linguagem Python com base nas aulas do professor Luciano Vasconcelos do Bootcamp Intensivo de Python da Jornada de Dados.

Portanto, dentro do repositório, você encontrará pastas divididas por cada aula e dentro de cada aula haverá arquivos com exemplos, exercícios e/ou desafios propostos pelo professor.
